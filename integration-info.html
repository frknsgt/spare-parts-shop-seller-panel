API Bilgilerini Güncelle

Entegrasyon bilgilerinizde güncelleme işlemi yapmanız durumunda, API kullanıcı adınız ve şifreniz değişecektir. Değişen bilgileri entegrasyon ekibiniz ya da entegrasyonunuzu sağlayan firmanız ile paylaşmanız gerekmektedir. Aksi durumda Yedek Parçacı Dünyası sistemiyle olan bağlantınız kopacağı için entegrasyon üzerinden yaptığınız işlemler Yedek Parçacı Dünyası sistemine yansımayacaktır.
